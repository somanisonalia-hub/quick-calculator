## SEO Strategy for Each Calculator

### For Every Calculator, Do This:

1. **Keywords Research First** (before building)
   - Primary keyword (high volume, medium competition)
   - Long-tail keywords (lower volume, lower competition, higher conversion)
   - Featured snippet opportunity keywords

2. **URL Structure**
   ```
   /calculators/calculator-name
   /tools/calculator-name
   ```

3. **Meta Tags**
   - `seoTitle`: Include primary keyword + brand (60 chars)
   - `metaDescription`: Include keyword + benefit (155 chars)
   - `keywords`: 5 primary keywords
   - `longTailKeywords`: 5 long-tail keywords

4. **Content Structure** (seoContent object)
   ```
   - introduction (why this calculator matters)
   - benefits (user value)
   - steps (how to use)
   - inputsExplained (what each field means)
   - formulaExplanation (the math, credibility)
   - examples (real-world scenarios)
   - resultsExplanation (interpret results)
   - whoItsFor (target audience)
   - disclaimer (legal protection)
   - relatedTools (internal links)
   - FAQs (answer common questions)
   ```

5. **Internal Linking Strategy**
   - Link from home page category section
   - Link from category pages
   - Link from related calculator FAQs
   - Create content clusters (related calculators linked together)

6. **Multi-language Support**
   - All 4 languages: en, es, pt, fr
   - Translated keywords matter for non-English ranking

## AdSense Optimization Tips

### 1. **Ad Placement Strategy**
   - Above calculator (responsive unit)
   - Below main input (vertical unit)
   - Below results (responsive unit)
   - Sidebar (desktop only)
   - Between related calculators

### 2. **Traffic Quality Matters**
   - Higher CPC for finance/real estate/health topics
   - Lower CPC for general education topics
   - Target high-intent searches (people solving problems = more ad clicks)

### 3. **Increase Page Views Per Session**
   - Related calculator suggestions after results
   - "You might also like" sections
   - Internal linking strategy
   - Keep users on site longer = more ad impressions

### 4. **Optimize for Mobile**
   - 60%+ traffic is mobile
   - Sticky footer ads work better on mobile
   - Ensure calculator is responsive

### 5. **High-Value Keywords**
   - Finance: mortgage, loan, investment ($2-5 CPC)
   - Real Estate: property, home, rent ($2-4 CPC)
   - Legal: will, estate, lawsuit ($3-8 CPC)
   - Home Improvement: paint, roof, wood ($1.5-3 CPC)
   - Insurance: quotes, coverage ($2-5 CPC)
   - Health (specific): pregnancy, fertility ($1-2.50 CPC)
   - General Health: BMI, exercise ($0.2-0.8 CPC)
